# File Automation Project

## Description
This project is a Python-based file automation script that performs file management operations automatically. It can organize files, rename files, delete files, and maintain logs of all operations performed.

## Objective
The main objective of this project is to automate repetitive file handling tasks using Python modules like OS and shutil.

## Features
- Automatic file sorting based on file extensions
- File renaming support
- File deletion support
- Exception handling for errors
- Operation logging using Python logging module

## Technologies Used
- Python
- OS Module
- Shutil Module
- Exception Handling
- Logging Module
